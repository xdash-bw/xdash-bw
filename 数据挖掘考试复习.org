1. 什么是数据挖掘？数据挖掘的步骤？
   - 步骤
     1. 数据收集
     2. 数据预处理
        1. 数据清洗
        2. 数据集成
        3. 数据转换
        4. 数据规约
     3. 数据挖掘
     4. 模式评估与知识表示

2. 简述数据预处理的方法和内容？
   1. 数据清洗
      1. 填充缺失的数据
      2. 光滑噪声数据
      3. 识别和删除离群点且保持一致性以清除数据
   2. 数据集成
      把多个不同的数据源集合在一起
   3. 数据转换
      把数据转换成可以挖掘的形式
      通过
      1. 平滑聚集
      2. 概化
      3. 规范化
   4. 数据规约
      把数据转换成规约表，减少数据量

3. 简述 KNN 算法的步骤？
   1. 选择参数 k
   2. 计算 *未知实例* 与 *所有已知实例* 的距离
      : 可以选择多种计算距离的方式
   3. 选择最近 K 个已知实例
   4. 根据少数服从多数的投票法则，让未知实例归类为 k 个最邻近样本种最多数的类别。

4. 朴素贝叶斯的思想
    朴素贝叶斯是非常简单的分类方法，对于给出的待分类项，计算待项发生的条件下，各个类别出现的概率，哪个最大，就认为此待分类项属于哪个类别。

5. 聚类定义？与分类的区别
   - 定义
    把对象集合分为由相似的对象组合而成的多个类的过程叫做聚类
   - 区别
    分类是属于监督学习，训练数据中包括想要预测的属性，即目标变量，从已标志类别的数据学习，去预测未标志的数据。

    聚类属于无监督学习，训练数据中没有相应的目标变量，只是为了发掘数据中相似样本的分组

6. 主成分分析的主要思想
    主成分分析也叫主分量分析，旨在利用降维的思想，把多指标转换为少数几个综合指标。在统计学中，主成分分析是一种简化数据集的技术。

7. 简述 ID3 算法的基本思想及其算法的步骤
   - 基本思想
     1. ID3 是一种决策树学习算法
     2. 在树中的每个节点使用信息增益作为判断基进行分类
     3. 使测试时每个非叶子节点获得最大的分类类别增益，使分类后的数据集的熵最小。

   - 步骤
     1. 先计算样本集中所有对象属性
     2. 把信息增益最大的属性作为测试属性，与测试属性相同的对象组成子样本
     3. 如果该样本集只有一种属性，那么就是作为叶子节点并且回溯。否则重复该算法。

8. 关联规则挖掘的挖掘过程
   1. 从事务集合中找出频繁项目集；
   2. 从频繁项目集合中生成满足最低置信度的关联规则。

9. 什么是离群点？简述离群点的类型和常用的检测方法？
   - 定义
     离群点是指在全局或者局部范围内偏离一般水平的观测对象
   - 类型
     全局，条件，集合
   - 检测方法
     1. 基于统计学的离群点检测
     2. 基于分类的离群点检测
     3. 基于聚类的离群点检测
     4. 基于邻近的离群点检测

10. 什么是协同过滤？协同过滤推荐系统思路？
    - 定义
      对于推荐系统来说，通过用户的持续协同作用，最终给用户的推荐会越来越准。 而过滤，就是从可行的决策(推荐)方案(标的物)中将用户喜欢的方案(标的物)找(过滤)出来。 具体来说，协同过滤的思路是通过群体的行为来找到某种相似性(用户之间的相似性或者标的物之间的相似性)，通过该相似性来为用户做决策和推荐。

    - 设计思路
      就讲那两种思路好了，简短点。
      - 基于用户的协同过滤
      - 基于物品的协同过滤
